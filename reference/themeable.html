themeable

themes.themeable.themeable(self, theme_element=None)

Notes

A user should never create instances of class themeable or subclasses of it.

Attributes

Name Description
order int([x]) -> integer
rcParams Return themeables rcparams to an rcparam dict before plotting.

apply

themes.themeable.themeable.apply(self, theme)

Called by the theme to apply the themeable

Subclasses shouldn’t have to override this method to customize.

apply_ax

themes.themeable.themeable.apply_ax(self, ax)

Called after a chart has been plotted.

Subclasses can override this method to customize the plot according to the theme.

Parameters

Name Type Description Default
ax matplotlib.axes.Axes required

This method should be implemented as super(…).apply_ax() followed by extracting the portion of the axes specific to this themeable then applying the properties.

apply_figure

setup_figure

themes.themeable.themeable.setup_figure(self, figure)

Apply theme to the figure

Compared to apply_figure, this method is called before any plotting is done. This is necessary in some cases where the drawing functions need(or can make use of) this information.
