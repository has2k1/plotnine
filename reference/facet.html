facet

facets.facet.facet(self, scales='fixed', shrink=True, labeller='label_value', as_table=True, drop=True, dir='h')

Base class for all facets

Parameters

Name Type Description Default
scales str in ['fixed', 'free', 'free_x', 'free_y'] Whether x or y scales should be allowed (free) to vary according to the data along the rows or the columns. Default is 'fixed'. 'fixed'
shrink bool Whether to shrink the scales to the output of the statistics instead of the raw data. Default is True. True
labeller str | function How to label the facets. If it is a str, it should be one of 'label_value' 'label_both' or 'label_context'. Default is 'label_value' 'label_value'
as_table bool If True, the facets are laid out like a table with the highest values at the bottom-right. If False the facets are laid out like a plot with the highest value a the top-right. Default it True. True
drop bool If True, all factor levels not used in the data will automatically be dropped. If False, all factor levels will be shown, regardless of whether or not they appear in the data. Default is True. True
dir str in ['h', 'v'] Direction in which to layout the panels. h for horizontal and v for vertical. 'h'
