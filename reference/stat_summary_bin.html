stat_summary_bin

stats.stat_summary_bin()

Summarise y values at x intervals

Usage

stat_summary_bin(mapping=None, data=None, geom='pointrange',
                 position='identity', na_rm=False, fun_ymax=None, bins=30,
                 fun_y=None, boundary=None, fun_args=None, fun_ymin=None,
                 binwidth=None, breaks=None, random_state=None, fun_data=None,
                 **kwargs)

Only the mapping and data can be positional, the rest must be keyword arguments. **kwargs can be aesthetics (or parameters) used by the geom.

Parameters

Name Type Description Default
mapping aes Aesthetic mappings created with aes. If specified and :py:inherit.aes=True, it is combined with the default mapping for the plot. You must supply mapping if there is no plot mapping.
Aesthetic Default value
x
y
The bold aesthetics are required. 		required

Notes

The binwidth, bins, breaks and bounary arguments can be a tuples with two values ((xaxis-value, yaxis-value)) of the required type.
